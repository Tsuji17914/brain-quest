# 仕様書

## コンセプト

冒険者ギルドに出勤し、今日の依頼書を選んで脳トレをこなす RPG 型の脳トレアプリ。

## 毎日の流れ

1. ログイン
2. クエストボードで本日の依頼書を確認
3. 難易度を選んで受注
4. 問題を解く（2〜3分）
5. 結果表示 → EXP・ゴールド獲得
6. レベルアップ・ストリーク更新

## ジャンルと回答形式

| ジャンル | 回答形式 |
|----------|----------|
| 記憶力 | 数列・並びを見て入力（再現） |
| 計算 | 数字をタイプ入力 |
| 言葉パズル | 選択式（4択）または並び替え |

- 毎日1ジャンル固定（AIが生成）
- 同じジャンルを何度でも練習可能

## 難易度と報酬

| 難易度 | 初回EXP | 2回目以降EXP | ゴールド |
|--------|---------|-------------|--------|
| ★ 易しい | 50 | 30 | 10G |
| ★★ 普通 | 90 | 60 | 20G |
| ★★★ 難しい | 150 | 100 | 35G |

## レベル・称号

| レベル | 称号 |
|--------|------|
| Lv1 | 見習い冒険者 |
| Lv5 | 駆け出し冒険者 |
| Lv15 | 一人前の冒険者 |
| Lv30 | 熟練冒険者 |
| Lv50 | 上級冒険者 |
| Lv75 | 英雄 |
| Lv99 | 伝説の賢者 |

## ストリーク

- 毎日クエストをこなすと連続記録が増える
- 1日でも空くとリセット
- 聖水アイテムで1回だけ保護可能

## ショップ・アイテム

| アイテム | 効果 | 値段 |
|----------|------|------|
| 聖水 | ストリーク1日保護 | 50G |
| 賢者の石 | 次回EXP2倍 | 80G |
| ヒント書 | 問題のヒントを表示 | 15G |

## AI の役割

- Claude API を使って毎日の問題を生成
- ジャンル × 難易度に応じたプロンプトで生成
- 前日の結果をもとに問題の傾向を調整（将来拡張）
